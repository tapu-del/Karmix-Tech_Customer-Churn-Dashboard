# Karmix-Tech_Customer-Churn-Dashboard


## 📊 Overview
Dashboard showing customer churn analysis for a telecom company.

## 📈 Key Metrics
- **Total Customers:** 7,043
- **Churned Customers:** 1,869
- **Churn Rate:** 26.54%
- **Active Customers:** 5,174

## 🔑 Key Insight
26.54% of customers are leaving the company.

## 🛠️ Tools Used
- Excel (Data Cleaning)
- Power BI (Dashboard)
